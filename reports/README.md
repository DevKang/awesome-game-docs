# 중간고사 대체 과제 - 게임 제안서
## 작성 방법
1. Repository Fork
2. 자신이 해당하는 폴더안에 위치한 READMD.md 파일 수정
3. Pull Request 작성
4. [이슈 페이지](https://github.com/DevKang/dgu-3d-script-docs/issues/1)에서 레퍼런스 추가하여 코멘트 달기

## 제출 후 수정 방법
1. 위에서 Fork한 리포지토리에서 새로운 Pull Request를 통해 원본 리포지토리의 수정 사항을 내 계정의 리포지토리에 반영
2. 위의 2~4번 다시 수행
